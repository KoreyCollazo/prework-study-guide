CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Think of margin and padding as overlapping rectangles

Git

  • When making a new branch, if on opening there are no files try to delete the branch and make a new one.
  • git status: checks the current branch being working on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout -D branch-name: deletes branch
  • git checkout branch-name: to select the branch to move to
  1. Confirm you are on the main branch with (git status) then follow
  2. git checkout main
  3. git checkout -b feature/(feature name)
  1. Confirm you are on the current feature branch with (git status) then follow
  2. git add -A (adds all files)
  3. git commit -m "brief description"
  4. git pull origin main
  5. git push origin feature/(current feature branch)
  6. Navigate to github and click the "Pull Requests" tab and then the "New pull request" button.
  7. Make sure base and comparison branch are correct and write a brief description of the changes
  8. Click the "create pull request" .
  9. Merge pull reaquest when ready

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
